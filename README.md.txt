# Booking API Automation Framework

This project contains automated tests for the Restful-Booker API using Rest Assured and TestNG.

## Prerequisites
- Java 11 or higher
- Maven 3.6 or higher

## Project Structure
- `src/main/java`: Contains the core framework code
- `src/test/java`: Contains the test cases
- `src/main/resources`: Contains configuration files
- `src/test/resources`: Contains test data
- `test-output`: Contains generated test reports

## Running Tests
```bash
mvn clean test
```

## Reports
Test reports can be found in the `test-output` directory after test execution.
